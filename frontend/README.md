# Takeat Orders Frontend

Chore inicial do frontend para o desafio Takeat.

## Objetivo deste chore

- Subir base React + Vite funcional
- Manter estrutura de pastas preparada para evolução
- Remover artefatos genéricos do template

## Stack

- React
- Vite
- ESLint

## Estrutura inicial

- `src/` ponto de entrada da aplicação
- `components/` componentes reutilizáveis (vazio por enquanto)
- `pages/` páginas da aplicação (vazio por enquanto)
- `routes/` configuração de rotas (vazio por enquanto)
- `services/` integração com API (vazio por enquanto)
- `styles/` estilos globais e utilitários (vazio por enquanto)

## Scripts

- `npm run dev` inicia frontend em desenvolvimento
- `npm run build` gera build de produção
- `npm run preview` serve build localmente
- `npm run lint` executa lint

## Execução local

```bash
npm install
npm run dev
```
